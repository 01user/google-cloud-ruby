Gcloud::Search::Result

Result¶ ↑

See Gcloud#search

Parent: Object
Methods: [] doc_id each fields names token

Methods

Public Instance Methods

[](name) ¶ ↑

Retrieve the field values associated to a field name.

Parameters¶ ↑

name

The name of the field. New values will be configured with this name. (String)

Returns¶ ↑

FieldValue

Example¶ ↑

require "gcloud"

gcloud = Gcloud.new
search = gcloud.search
index = search.index "products"

documents = index.search "best T-shirt ever"
document = documents.first
puts "The best match for your search is:"
document["description"].each do |value|
  puts "* #{value} (#{value.type}) [#{value.lang}]"
end

doc_id() ¶ ↑

The unique identifier of the document referenced in the search result.

each(&block) ¶ ↑

Calls block once for each field, passing the field name and values pair as parameters. If no block is given an enumerator is returned instead. (See Gcloud::Search::Fields#each)

Example¶ ↑

require "gcloud"

gcloud = Gcloud.new
search = gcloud.search
index = search.index "products"

documents = index.search "best T-shirt ever"
document = documents.first
puts "The best match for your search is:"
document.each do |name, values|
  puts "* #{name}:"
  values.each do |value|
    puts "  * #{value} (#{value.type})"
  end
end

fields() ¶ ↑

The fields in the search result. Each field has a name (String) and a list of values (FieldValues). (See Fields)

names() ¶ ↑

Returns a new array populated with all the field names. (See Gcloud::Search::Fields#names)

require "gcloud"

gcloud = Gcloud.new
search = gcloud.search
index = search.index "products"

documents = index.search "best T-shirt ever"
document = documents.first
puts "The best match has the following fields:"
document.names.each do |name|
  puts "* #{name}:"
end

token() ¶ ↑

The token for the next page of results.
