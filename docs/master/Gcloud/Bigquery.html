Gcloud::Bigquery

Google Cloud BigQuery¶ ↑

Google BigQuery enables you to query massive datasets without the time and expense of building hardware and infrastructure. BigQuery enables super-fast, SQL-like queries against append-only tables, using the processing power of Google's infrastructure.
