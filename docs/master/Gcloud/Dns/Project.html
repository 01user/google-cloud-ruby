Gcloud::Dns::Project

Project¶ ↑

The project is a top level container for resources including Cloud DNS ManagedZones. Projects can be created only in the APIs console.

require "gcloud"

gcloud = Gcloud.new
dns = gcloud.dns
zone = dns.zone "example-zone"
zone.records.each do |record|
  puts record.name
end

See Gcloud#dns

Parent: Object

create_zone(zone_name, zone_dns, options = {}) ¶ ↑

Creates a new zone.

Parameters¶ ↑

zone_name

User assigned name for this resource. Must be unique within the project. The name must be 1-32 characters long, must begin with a letter, end with a letter or digit, and only contain lowercase letters, digits or dashes. (String)

zone_dns

The DNS name of this managed zone, for instance “example.com.”. (String)

options

An optional Hash for controlling additional behavior. (Hash)

options[:description]

A string of at most 1024 characters associated with this resource for the user's convenience. Has no effect on the managed zone's function. (String)

options[:name_server_set]

A NameServerSet is a set of DNS name servers that all host the same ManagedZones. Most users will leave this field unset. (String)

Returns¶ ↑

Gcloud::Dns::Zone

Examples¶ ↑

require "gcloud"

gcloud = Gcloud.new
dns = gcloud.dns
zone = dns.create_zone "example-zone", "example.com."

You can also pass description and name_server_set options.

require "gcloud"

gcloud = Gcloud.new
dns = gcloud.dns
zone = dns.create_zone "example-zone", "example.com.",
                        description: "A description of my zone.",
                        name_server_set: "example"

project() ¶ ↑

The DNS project connected to.

Example¶ ↑

require "gcloud"

gcloud = Gcloud.new "my-todo-project", "/path/to/keyfile.json"
dns = gcloud.dns

dns.project #=> "my-todo-project"

zone(zone_id) ¶ ↑

Retrieves an existing zone by name or id.

Parameters¶ ↑

zone_id

The name or id of a zone. (String or Integer)

Returns¶ ↑

Gcloud::Dns::Zone or nil if the zone does not exist

Example¶ ↑

require "gcloud"

gcloud = Gcloud.new
dns = gcloud.dns
zone = dns.zone "example-zone"
puts zone.name

zones(options = {}) ¶ ↑

Retrieves the list of zones belonging to the project.

Parameters¶ ↑

options

An optional Hash for controlling additional behavior. (Hash)

options[:token]

A previously-returned page token representing part of the larger set of results to view. (String)

options[:max]

Maximum number of zones to return. (Integer)

Returns¶ ↑

Array of Gcloud::Dns::Zone (Gcloud::Dns::Zone::List)

Examples¶ ↑

require "gcloud"

gcloud = Gcloud.new
dns = gcloud.dns
zones = dns.zones
zones.each do |zone|
  puts zone.name
end

If you have a significant number of zones, you may need to paginate through them: (See Gcloud::Dns::Zone::List)

require "gcloud"

gcloud = Gcloud.new
dns = gcloud.dns
zones = dns.zones
loop do
  zones.each do |zone|
    puts zone.name
  end
  break unless zones.next?
  zones = zones.next
end
