Public Instance Methods

ack!()

Alias for: acknowledge!

ack_id()

The acknowledgment ID for the message being acknowledged. This was returned by the Pub/Sub system in the Pull response. This ID must be used to acknowledge the received event.

acknowledge!()

Acknowledges receipt of the message.

Also aliased as: ack!

delay!(new_deadline)

Modifies the acknowledge deadline for the message. This method is useful to indicate that more time is needed to process the message, or to make the message available for redelivery if the processing was interrupted.

message()

The received message.

Also aliased as: msg
